Color Clicker Links werden 16 farbige Ziele dargestellt. Eines der Ziele unterscheidet sich farblich etwas von den übrigen Elementen. Finde dieses Element und klicke es an. Für jedes korrekt identifizierte Element gibt es einen Punkt. Mit jeder Runde wird es schwieriger.

Das Spiel ist eine neue Implementierung dieser Idee von Peter Lauris.
